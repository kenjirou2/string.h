# string.h

![C Language](https://img.shields.io/badge/language-C-blue?logo=c) 
![Windows](https://img.shields.io/badge/platform-Windows-lightgrey?logo=windows) 
![Linux](https://img.shields.io/badge/platform-Linux-orange?logo=linux)



string.h is a header file for **string manipulation** in C. It provides declarations for functions commonly used to handle strings, such as copying, concatenation, comparison, and searching for characters or substrings.

---

## Installation
---
Clone the repo:



```
	git clone https://github.com/kenjirou2/string.h.git
```

include the file header in your C/Cpp project:



```
	#include "string.h"
```
